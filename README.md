## About

I build voice-first, accessible technology. I'm the founder and president of
[Third Eye Worldwide](https://teww.org), where we make free, open-source
assistive tools for people with visual impairment — designed from one premise:
a blind user is the default user, not an afterthought.

**Roles:** Founder & President, Third Eye Worldwide · Member, [PIIIX](https://github.com/PIIIX-org).

## Why this profile is built the way it is

I work on accessibility, so this page follows the same rules I hold my products to:

- **Reduced motion is respected.** The header animation stops on its own if your system asks for reduced motion.
- **Every image has a text description**, so a screen reader can convey it.
- **High contrast, real headings, and links that say where they go** — no "click here."
- **Mostly text on purpose.** Text is what everyone can read, including the screen-reader users I build for.

If any part of this profile is hard to read or use, that's a bug — tell me and I'll fix it.
