## How I like to build

- Start with the simplest architecture that can solve the real problem.
- Prefer accessible, native browser capabilities before adding custom runtime code.
- Keep decisions and calculations visible instead of hiding them behind magic numbers.
- Treat security tooling as defensive, authorized, and safe by default.
- Write documentation that helps the next person run, understand, and improve the project.
